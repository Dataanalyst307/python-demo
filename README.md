# Notes App Mockup

A simple personal Notes app interface built as a single responsive HTML file with inline CSS.

## Overview

This project is a frontend-only mockup inspired by Apple's clean and minimal design style. It includes:

- A top header with the app title, search field, and profile avatar
- A folders sidebar with a "New Note" button
- A notes list with realistic preview cards
- A reading panel showing a selected note in full

## Tech Stack

- HTML
- CSS

No JavaScript, frameworks, libraries, Bootstrap, or Tailwind are used.

## Design Goals

- Clean and premium visual style
- Neutral color palette
- Soft spacing and rounded corners
- Subtle borders and shadows
- Responsive layout for mobile, tablet, and desktop
- Inter font throughout the interface

## File Structure

- `index.html` - complete Notes app UI in one file

## How to Use

Open `index.html` in any modern browser to view the mockup.

## Notes

This is a static UI demo with placeholder content only. Search, folder selection, and note switching are visual elements and do not have interactive behavior.
